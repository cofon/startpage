# 起始页扩展

这是一个配合起始页使用的浏览器扩展，主要用于快速添加网站到起始页，以及将新标签页设置为起始页。

## 功能

1. **快速添加网站**：在任意标签页点击扩展图标，自动获取当前页面的标题、描述和图标，快速添加到起始页
2. **新标签页重定向**：打开新标签页时自动重定向到起始页
3. **支持离线添加**：当起始页未打开时，数据会保存在扩展存储中，等起始页打开后自动同步
4. **支持多种编码**：解决中文网站乱码问题
5. **多级图标获取**：提高图标获取成功率

## 安装方法

1. **打开 Edge 浏览器**
2. **进入扩展管理页面**：在地址栏输入 `edge://extensions/`
3. **启用开发者模式**：点击页面右上角的"开发者模式"开关
4. **加载扩展**：点击"加载已解压的扩展程序"，选择本目录（`extension` 文件夹）
5. **确认安装**：扩展应该会出现在扩展列表中

## 使用方法

### 添加网站

1. **访问任意网站**
2. **点击扩展图标**：在浏览器工具栏中找到扩展图标，点击它
3. **查看自动填充**：扩展会自动填充网站的标题、描述和图标
4. **确认添加**：点击"确认添加"按钮，网站会被添加到起始页

### 新标签页

- **自动重定向**：打开新标签页时，会自动重定向到起始页（`https://dgf.cc.cd`）
- **开发环境**：如果需要在开发环境测试，可以修改 `newtab.html` 文件中的 URL 为 `http://localhost:5173`

### 键盘快捷键

- **Alt+Shift+D**：快速打开添加网站面板

## 技术说明

### 文件结构

```
extension/
├── background.js         # 扩展后台脚本
├── content.js            # 扩展内容脚本
├── manifest.json         # 扩展配置
├── newtab.html           # 新标签页重定向页面
├── popup.html            # 扩展弹出面板
├── popup.js              # 扩展弹出面板脚本
└── README.md             # 本说明文档
```

### 数据存储

- **扩展存储**：使用 `chrome.storage.local` 存储未同步的网站元数据
- **起始页存储**：数据最终会存储在起始页的 IndexedDB 中

### 通信机制

- **扩展 → 起始页**：通过消息传递机制将网站元数据发送给起始页
- **起始页 → 扩展**：起始页可以请求扩展同步未处理的网站元数据

## 注意事项

1. **权限说明**：扩展需要以下权限
   - `activeTab`：访问当前标签页
   - `scripting`：在网页上执行脚本
   - `tabs`：管理标签页
   - `storage`：使用本地存储
   - `offscreen`：创建离屏文档

2. **域名配置**：扩展只与以下域名通信
   - `http://localhost:5173/`
   - `http://localhost:5174/`
   - `http://localhost:5175/`
   - `https://dgf.cc.cd/`
   - `http://dgf.cc.cd/`

3. **开发环境**：在开发起始页时，可以修改 `newtab.html` 文件中的 URL 为 `http://localhost:5173`

4. **自定义**：如果需要修改重定向的起始页 URL，只需修改 `newtab.html` 文件中的 `window.location.href` 值

## 故障排除

- **添加失败**：检查起始页是否打开，或者是否有权限访问起始页
- **图标获取失败**：有些网站可能没有图标，或者图标获取失败，这是正常的
- **新标签页不重定向**：检查扩展是否正确安装，以及 `newtab.html` 文件是否存在
